To enable the "Download CV" link in the header, place your CV PDF file here with the exact filename:

  CV_NazeeraNasharuddin_2026.pdf

Path from project root:
  frontend/portfolio/public/CV_NazeeraNasharuddin_2026.pdf

Once you add the file the header's "Download CV" link will open/download it automatically.

If you prefer a different filename, update `src/components/Header.js` and change the `href` for the Download CV item to the desired filename (starting with a leading slash, e.g. `/my-cv.pdf`).
